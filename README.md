**CoreFlux Lite** is an internal workflow tool that helps technicians and engineers quickly understand system and customer context without digging through fragmented ticket history.

---
#### The Problem:

In many support environments, critical information is scattered across multiple tickets or stuck in teams chats. (no really).

#### So what Can It Do?

- Search systems or customers  
- Instantly view recent activity and ownership information 
- Identify recurring issues and patterns  
- Access internal technical notes  
- Capture diagnostic context without duplicating ticket work  



### ⋆˙⟡ Features  ⋆˙⟡
---
System Search
- Lookup by system ID  
- View current status, location, and ownership  
- See last technician and last action performed  
- Highlight recurring issue patterns  

Customer Search
- Lookup by customer or team  
- View preferences and configuration standards  
- See systems under their ownership  
- View recent system activity  

---

#### Internal Technical Notes
- Append-only notes tied to systems or customers  
- Used for diagnosis, resolution context, and handoff  
- Scrollable history for quick reference  
- Separated from customer-facing 

---

#### Dual-Output Notes
- Write a note once  
- Option to generate a customer-safe ticket update version 
- Reduces redundant work between internal notes and ticket updates  

---

#### Pattern Detection
- Identifies repeated system issues  
- Surfaces trends such as:
  - boot failures  
  - hardware replacements  
  - recurring configuration issues
  - recurring connectivity issues 

---

#### Tech Stack
- Backend: Python, FastAPI  
- Database: SQLite  
- Frontend: HTML, CSS, JavaScript
